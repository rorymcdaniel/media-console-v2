  <name>Task 1: Create CMake build system with Qt6, code quality configs, and pre-commit hook</name>
  <files>CMakeLists.txt, .clang-format, .clang-tidy, .githooks/pre-commit, .gitignore</files>

Create the top-level CMakeLists.txt:
- cmake_minimum_required(VERSION 3.16)
- project(media-console VERSION 2.0.0 LANGUAGES CXX)
- set CMAKE_CXX_STANDARD 17, CMAKE_CXX_STANDARD_REQUIRED ON, CMAKE_AUTOMOC ON, CMAKE_AUTORCC ON
- find_package(Qt6 REQUIRED COMPONENTS Core Gui Quick Qml) -- minimal set for Phase 1, expanded later
- qt_standard_project_setup(REQUIRES 6.8)
- clang-tidy integration: find_program(CLANG_TIDY_EXE NAMES clang-tidy), if found set CMAKE_CXX_CLANG_TIDY
- Version extraction from git tags using execute_process(git describe) -- lift pattern from original CMakeLists.txt lines 154-215
- Pass VERSION_STRING, VERSION_MAJOR, VERSION_MINOR, VERSION_PATCH, GIT_COMMIT as compile definitions
- qt_add_executable(media-console src/main.cpp src/utils/Logging.h src/utils/Logging.cpp)
- qt_add_qml_module(media-console URI MediaConsole VERSION 1.0 QML_FILES src/qml/main.qml)
- target_include_directories(media-console PRIVATE ${CMAKE_SOURCE_DIR}/src)
- target_link_libraries(media-console PRIVATE Qt6::Core Qt6::Gui Qt6::Quick Qt6::Qml)
- option(BUILD_TESTS "Build unit tests" ON) with enable_testing() and add_subdirectory(tests) placeholder
- Do NOT include platform-specific library linking yet (no ALSA, libgpiod, etc.) -- that comes in later phases

Create .clang-format (per CONTEXT.md locked decisions):
- BasedOnStyle: WebKit
- ColumnLimit: 120
- IndentWidth: 4
- BreakBeforeBraces: Allman
- IncludeBlocks: Regroup with categories: corresponding header (Priority 1), Qt headers matching regex '^<Q' (Priority 2), system headers matching '^<' (Priority 3), project headers matching '^"' (Priority 4)
- SortIncludes: CaseSensitive
- AccessModifierOffset: -4
- AlignAfterOpenBracket: Align
- AllowShortFunctionsOnASingleLine: Inline
- SpaceAfterTemplateKeyword: true

Create .clang-tidy:
- Checks: -*, bugprone-*, modernize-*, performance-*, readability-*, -modernize-use-trailing-return-type, -readability-redundant-access-specifiers, -readability-identifier-length
- HeaderFilterRegex: 'src/.*\.h$'
- WarningsAsErrors: '' (empty -- warnings only, does not block build)

Create .githooks/pre-commit:
- Bash script that finds staged .cpp/.h/.hpp files via git diff --cached
- Runs clang-format -i on each staged file
- Re-stages formatted files with git add
- Exits 0 (format only, no blocking)
- Make executable (chmod +x)

Create .gitignore:
- build/, cmake-build-*/, .cache/, *.o, *.so, *.dylib, moc_*, qrc_*, .qmlc, CMakeFiles/, Testing/

Configure git to use .githooks directory:
- Run: git config core.hooksPath .githooks
