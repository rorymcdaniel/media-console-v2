    - toHexCode(MediaSource::Bluetooth) returns 0x2E

    - toHexCode(MediaSource::None) returns 0x00
    - fromHexCode(0x2B) returns MediaSource::Streaming

    - fromHexCode(0x23) returns MediaSource::CD (ambiguous with Library, CD is default)
    - fromHexCode(0x05) returns MediaSource::Computer
    - fromHexCode(0x2E) returns MediaSource::Bluetooth
    - fromHexCode(0xFF) returns MediaSource::None (unknown code)
    - fromHexCode(0x00) returns MediaSource::None
