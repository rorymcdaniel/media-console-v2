eISCP Protocol (from research):
- Header: "ISCP" (4 bytes) + headerSize=16 (4 bytes BE) + messageSize (4 bytes BE) + version=1 (1 byte) + reserved (3 bytes)
- ISCP payload: "!1" + command + "\r"
- Total packet = 16-byte header + ISCP payload
- Response parsing: strip "!1" prefix and trailing CR/LF/EOF (0x1A) bytes

// EiscpMessage.h
#pragma once
#include <QByteArray>
#include <QString>

class EiscpMessage
{
public:
    /// Build an eISCP packet from an ISCP command string (e.g., "PWRQSTN", "MVL1A").
    /// Returns the complete binary packet ready to send over TCP.
    static QByteArray build(const QString& command);

    /// Attempt to parse one complete eISCP packet from the front of buffer.
    /// On success: returns the ISCP command string (e.g., "MVL1A"), removes consumed bytes from buffer.
    /// On failure (incomplete data): returns empty QString, buffer unchanged.
    /// On corruption (invalid magic): clears buffer, returns empty QString.
    static QString parse(QByteArray& buffer);

private:
    static constexpr int kHeaderSize = 16;
    static constexpr char kMagic[] = "ISCP";
    static constexpr quint8 kVersion = 1;
};
```

Implementation in EiscpMessage.cpp:
- build(): Construct ISCP payload as "!1" + command.toLatin1() + "\r". Build 16-byte header: "ISCP" (4) + qToBigEndian(quint32(16)) (4) + qToBigEndian(quint32(payload.size())) (4) + version=1 (1) + 3 zero bytes. Concatenate header + payload.
- parse(): Check buffer.size() >= 16. Validate magic == "ISCP". Extract messageSize from bytes 8-11 using qFromBigEndian<quint32>. Check buffer.size() >= 16 + messageSize. Extract ISCP payload from buffer.mid(16, messageSize). Remove consumed bytes with buffer.remove(0, 16 + messageSize). Strip "!1" prefix. Strip trailing '\r', '\n', and char(0x1A). Return cleaned command string.
- If magic is invalid, call buffer.clear() and return empty.

Use QDataStream or manual qToBigEndian/qFromBigEndian for byte order. Prefer qToBigEndian for simplicity.

#pragma once
#include <QObject>
#include <QTcpSocket>
#include <QTimer>

class EiscpConnection : public QObject
{
    Q_OBJECT

public:
    explicit EiscpConnection(QObject* parent = nullptr);

    /// Start connecting to the receiver. Will auto-reconnect on failure/disconnect.
    void connectToReceiver(const QString& host, int port);

    /// Disconnect and stop auto-reconnect.
    void disconnect();

    /// Send an ISCP command (e.g., "MVL1A"). Frames as eISCP and writes to socket.
    /// No-op if not connected.
    void sendCommand(const QString& command);

    /// Returns true if the TCP socket is in Connected state.
    bool isConnected() const;

signals:
    void connected();
    void disconnected();
    void messageReceived(const QString& message);
    void connectionError(const QString& error);

private slots:
    void onConnected();
    void onDisconnected();
    void onReadyRead();
    void onSocketError(QAbstractSocket::SocketError error);
    void onReconnectTimer();

private:
    void resetBackoff();

    QTcpSocket m_socket;
    QTimer m_reconnectTimer;
    QByteArray m_readBuffer;
    QString m_host;
    int m_port = 0;
    int m_currentBackoffMs = 1000;
    bool m_autoReconnect = true;

    static constexpr int kInitialBackoffMs = 1000;
    static constexpr int kMaxBackoffMs = 30000;
};
```

Implementation in EiscpConnection.cpp:
- Constructor: Wire QTcpSocket signals (connected, disconnected, readyRead, errorOccurred) to private slots. Wire m_reconnectTimer::timeout to onReconnectTimer. Set m_reconnectTimer to single-shot. Use mediaReceiver logging category.
- connectToReceiver(): Store host/port. Set m_autoReconnect = true. Call m_socket.abort() first (safety), then m_socket.connectToHost(host, port). Log: qCInfo(mediaReceiver) << "Connecting to" << host << ":" << port;
- disconnect(): Set m_autoReconnect = false. m_reconnectTimer.stop(). m_socket.abort().
- sendCommand(): If socket state != ConnectedState, return silently. Build packet via EiscpMessage::build(command). Write to socket. Log at debug level.
- isConnected(): return m_socket.state() == QAbstractSocket::ConnectedState.
- onConnected(): resetBackoff(). m_reconnectTimer.stop(). Log info. Emit connected().
- onDisconnected(): If m_autoReconnect, start reconnect timer with current backoff. Log info. Emit disconnected().
- onReadyRead(): Append m_socket.readAll() to m_readBuffer. Loop: parse = EiscpMessage::parse(m_readBuffer); if empty, break. Else emit messageReceived(parse). Log at debug level.
- onSocketError(): Log warning with error string. Emit connectionError(m_socket.errorString()). If m_autoReconnect and m_reconnectTimer not active, start it.
- onReconnectTimer(): Log info with backoff value. m_currentBackoffMs = qMin(m_currentBackoffMs * 2, kMaxBackoffMs). m_socket.abort(). m_socket.connectToHost(m_host, m_port).
- resetBackoff(): m_currentBackoffMs = kInitialBackoffMs.
