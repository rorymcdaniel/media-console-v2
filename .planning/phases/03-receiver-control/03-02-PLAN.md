eISCP command/response format (from research):
- MVL{hex}: volume, hex 00-C8 (0-200). MVLQSTN to query.
- PWR{00|01}: power off/on. PWRQSTN to query.
- AMT{00|01|TG}: mute off/on/toggle. AMTQSTN to query.
- SLI{hex}: input select. SLIQSTN to query.
- NTI{text}: track title
- NAT{text}: artist name
- NAL{text}: album name
- NJA{url} / NJA2{url}: album art URL
- NFI{text}: file info (format, bitrate)
- NTM{mm:ss/mm:ss}: time position/duration
- NMS{xx...}: network service, first 2 chars = service code (0A=Spotify, 04=Pandora, 18=AirPlay)
- NST{prs}: playback state, first char: P=playing, p=paused, S=stopped

Create src/receiver/ReceiverController.h:
```cpp
#pragma once
#include <QObject>
#include <QTimer>
#include <QElapsedTimer>

class EiscpConnection;
class ReceiverState;
class PlaybackState;
class UIState;
enum class MediaSource : int;

class ReceiverController : public QObject
{
    Q_OBJECT

public:
    ReceiverController(EiscpConnection* connection,
                       ReceiverState* receiverState,
                       PlaybackState* playbackState,
                       UIState* uiState,
                       QObject* parent = nullptr);

    /// Start connection and polling. Call after construction.
    void start(const QString& host, int port);

    /// Stop connection and polling.
    void stop();

public slots:
    void setVolume(int volume);
    void setPower(bool on);
    void selectInput(MediaSource source);
    void toggleMute();

    /// Query all receiver state (MVL, PWR, AMT, SLI). Called on connect and by poll timer.
    void queryAll();

private slots:
    void onConnected();
    void onDisconnected();
    void onMessage(const QString& message);
    void onPollTimer();
    void onStaleDataCheck();

private:
    void parseResponse(const QString& message);
    void clearMetadata();
    QString volumeToHex(int volume) const;
    int hexToVolume(const QString& hex) const;
    StreamingService detectService(const QString& code) const;
    QString serviceCodeToName(const QString& code) const;

    EiscpConnection* m_connection;
    ReceiverState* m_receiverState;
    PlaybackState* m_playbackState;
    UIState* m_uiState;

    QTimer m_pollTimer;
    QTimer m_staleDataTimer;
    QElapsedTimer m_lastMessageTime;
    bool m_staleWarningActive = false;

    static constexpr int kPollIntervalMs = 2500;
    static constexpr int kStaleDataThresholdMs = 30000;
    static constexpr int kStaleCheckIntervalMs = 5000;
};
```

Implementation in ReceiverController.cpp:
- Constructor: Store pointers. Wire EiscpConnection::connected -> onConnected, disconnected -> onDisconnected, messageReceived -> onMessage. Wire poll timer to onPollTimer (2500ms interval). Wire stale data timer to onStaleDataCheck (5000ms interval). Initialize m_lastMessageTime.
- start(): Call m_connection->connectToReceiver(host, port).
- stop(): m_pollTimer.stop(). m_staleDataTimer.stop(). m_connection->disconnect().
- setVolume(): Send "MVL" + volumeToHex(qBound(0, volume, 200)).
- setPower(): Send "PWR01" or "PWR00".
- selectInput(): clearMetadata() first. For non-streaming inputs (Phono, CD, Computer, Bluetooth), set receiverState title to the input name (e.g., "Phono", "CD"). For Streaming and Library, don't set placeholder. Then send "SLI" + hex code via toHexCode().
- toggleMute(): Send "AMTTG".
- queryAll(): Send MVLQSTN, PWRQSTN, AMTQSTN, SLIQSTN.
- onConnected(): m_uiState->setReceiverConnected(true). queryAll(). Start poll timer. Start stale data timer. Reset m_lastMessageTime.
- onDisconnected(): m_uiState->setReceiverConnected(false). Stop poll timer. Stop stale data timer. Do NOT clear ReceiverState (preserve last-known).
- onMessage(): m_lastMessageTime.restart(). If m_staleWarningActive, clear it. parseResponse(message).
- onPollTimer(): queryAll().
- onStaleDataCheck(): If playbackState->playbackMode() == Playing && m_lastMessageTime.elapsed() > kStaleDataThresholdMs, emit stale warning (set a property or signal). Otherwise if stale was active, clear it.
- parseResponse(): Extract 3-char command prefix. Route to handlers:
  - "MVL": hexToVolume on payload, set receiverState->setVolume.
  - "PWR": "01" -> setPowered(true), "00" -> setPowered(false).
  - "AMT": "01" -> setMuted(true), "00" -> setMuted(false).
  - "SLI": fromHexCode on 2-char hex payload, set receiverState->setCurrentInput.
  - "NTI": setTitle(payload after "NTI").
  - "NAT": setArtist(payload after "NAT").
  - "NAL": setAlbum(payload after "NAL").
  - "NJA": setAlbumArtUrl(payload after "NJA"). Also handle "NJA2" (4-char prefix: check for NJA2 first since NJA is a prefix of NJA2).
  - "NFI": setFileInfo(payload after "NFI").
  - "NMS": first 2 chars after "NMS" = service code. detectService() for enum. serviceCodeToName() for string. Set both.
  - "NST": first char after "NST": 'P' = Playing, 'p' = Paused, else Stopped. Set playbackState->setPlaybackMode.
  - "NTM": parse "mm:ss/mm:ss" into position and duration seconds, set playbackState.
- clearMetadata(): Set title, artist, album, albumArtUrl, fileInfo, serviceName to empty string. Set streamingService to Unknown. Set playbackMode to Stopped.
- volumeToHex(): return QString("%1").arg(qBound(0, volume, 200), 2, 16, QChar('0')).toUpper()
- hexToVolume(): bool ok; int val = hex.toInt(&ok, 16); return ok ? qBound(0, val, 200) : 0;
- detectService(): "0A" -> Spotify, "04" -> Pandora, "18" -> AirPlay. Others -> Unknown. Log unknown codes at info level.
- serviceCodeToName(): "0A" -> "Spotify", "04" -> "Pandora", "18" -> "AirPlay", "13" -> "iHeartRadio". Others -> "Unknown".

Stale data: ReceiverController should emit a signal `staleDataDetected(bool stale)` that can be connected to UI in a future phase. For now, just log it. The 30-second threshold is checked every 5 seconds for efficiency.
