  <behavior>
    PlaybackRouter:
    - play() dispatches to CdController::play(), FlacLibraryController::play(), or SpotifyController::play() based on PlaybackState::activeSource
    - pause(), stop(), next(), previous() dispatch similarly
    - seek(int ms) dispatches to local controllers (CD, Library); Spotify seek not implemented in Phase 8 controller
    - For Streaming source: call SpotifyController (if streamingService is Spotify) otherwise no-op
    - For Phono, Bluetooth, Computer, None: no local control, commands are no-op (receiver handles these)
    - When activeSource changes and old source was Playing, call stop() on old source's controller
    - CD vs Library disambiguation: both share receiver input 0x23. PlaybackRouter checks PlaybackState::activeSource (not ReceiverState::currentInput) for dispatch — activeSource is set explicitly by CdController or FlacLibraryController when they start playback
    - All dispatch methods are Q_INVOKABLE for QML binding in Phase 10

    AlbumArtResolver:
    - Q_PROPERTY QString albumArtUrl READ albumArtUrl NOTIFY albumArtUrlChanged
    - When activeSource is Streaming or Bluetooth: use ReceiverState::albumArtUrl (receiver CGI)
    - When activeSource is CD: use PlaybackState::albumArtUrl (set by CdController from local cache)
    - When activeSource is Library: use PlaybackState::albumArtUrl (set by FlacLibraryController from local cache)
    - When activeSource is Phono, Computer, None: return empty string (placeholder handled by QML)
    - Reacts to: PlaybackState::activeSourceChanged, PlaybackState::albumArtUrlChanged, ReceiverState::albumArtUrlChanged
    - Emits albumArtUrlChanged whenever the resolved URL changes
  </behavior>
  <action>
    Create src/orchestration/ directory.

    **PlaybackRouter.h:**
    ```cpp
    class PlaybackRouter : public QObject {
        Q_OBJECT
    public:
        PlaybackRouter(PlaybackState* playbackState,
                       ReceiverController* receiverController,
                       CdController* cdController,
                       FlacLibraryController* flacLibraryController,
                       SpotifyController* spotifyController,
                       QObject* parent = nullptr);

        Q_INVOKABLE void play();
        Q_INVOKABLE void pause();
        Q_INVOKABLE void stop();
        Q_INVOKABLE void next();
        Q_INVOKABLE void previous();
        Q_INVOKABLE void seek(int ms);

    private slots:
        void onActiveSourceChanged(MediaSource newSource);

    private:
        PlaybackState* m_playbackState;
        ReceiverController* m_receiverController;
        CdController* m_cdController;
        FlacLibraryController* m_flacLibraryController;
        SpotifyController* m_spotifyController;
        MediaSource m_previousSource = MediaSource::None;
    };
    ```

    - Constructor: connect PlaybackState::activeSourceChanged to onActiveSourceChanged
    - onActiveSourceChanged: if m_previousSource had an active controller and was Playing, call stop on that controller. Update m_previousSource.
    - Each command method: switch on m_playbackState->activeSource(), call correct controller. Use null-checks on all controller pointers (FlacLibraryController may be nullptr on non-Linux).
    - Log dispatch decisions: `qCInfo(mediaApp) << "PlaybackRouter: dispatching play() to" << sourceName;`

    **AlbumArtResolver.h:**
    ```cpp
    class AlbumArtResolver : public QObject {
        Q_OBJECT
        Q_PROPERTY(QString albumArtUrl READ albumArtUrl NOTIFY albumArtUrlChanged)
    public:
        AlbumArtResolver(PlaybackState* playbackState,
                         ReceiverState* receiverState,
                         QObject* parent = nullptr);

        QString albumArtUrl() const;

    signals:
        void albumArtUrlChanged(const QString& url);

    private slots:
        void resolve();

    private:
        PlaybackState* m_playbackState;
        ReceiverState* m_receiverState;
        QString m_albumArtUrl;
    };
    ```

    - Constructor: connect PlaybackState::activeSourceChanged, PlaybackState::albumArtUrlChanged, ReceiverState::albumArtUrlChanged all to resolve()
    - resolve(): Switch on activeSource. For Streaming/Bluetooth: use ReceiverState::albumArtUrl. For CD/Library: use PlaybackState::albumArtUrl. For others: empty string. If changed, update m_albumArtUrl and emit signal.
