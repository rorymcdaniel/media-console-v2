<task type="auto">
  <name>Task 3: Replace test harness main.qml with production layout skeleton</name>
  <files>src/qml/main.qml</files>
  <action>
Replace the entire test harness main.qml with the production layout. The layout must target 1920x720 (UI-03).

**Structure:**
```qml
import QtQuick
import QtQuick.Layouts
import QtQuick.Controls
import MediaConsole 1.0

Window {
    id: root
    width: 1920
    height: 720
    visible: true
    title: "Media Console"
    color: Theme.primaryBg

    // Global touch activity detection (UI-18)
    // This MouseArea covers the entire window and forwards all
    // press events to ScreenTimeoutController without consuming them.
    MouseArea {
        anchors.fill: parent
        z: 1000  // Above all content
        propagateComposedEvents: true
        acceptedButtons: Qt.AllButtons
        onPressed: (mouse) => {
            ScreenTimeoutController.activityDetected()
            mouse.accepted = false
        }
        onReleased: (mouse) => { mouse.accepted = false }
        onClicked: (mouse) => { mouse.accepted = false }
    }

    // Top status bar (UI-03)
    Rectangle {
        id: statusBar
        anchors.top: parent.top
        anchors.left: parent.left
        anchors.right: parent.right
        height: Theme.statusBarHeight
        color: Theme.secondaryBg
        z: 10

        RowLayout {
            anchors.fill: parent
            anchors.leftMargin: Theme.spacingMedium
            anchors.rightMargin: Theme.spacingMedium

            // Left: Error banner area (UI-15)
            Item {
                Layout.fillWidth: true
                Layout.fillHeight: true

                Text {
                    anchors.verticalCenter: parent.verticalCenter
                    text: "Receiver Disconnected"
                    color: Theme.errorColor
                    font.pixelSize: Theme.fontSizeBody
                    visible: !UIState.receiverConnected
                }
            }

            // Right side controls
            RowLayout {
                spacing: Theme.spacingMedium

                // Eject button placeholder (UI-13 — visible only when CD)
                Rectangle {
                    width: Theme.touchTargetSmall
                    height: Theme.touchTargetSmall
                    radius: Theme.radiusSmall
                    color: "transparent"
                    visible: PlaybackState.activeSource === MediaSource.CD

                    Text {
                        anchors.centerIn: parent
                        text: "\u23CF" // eject symbol
                        color: Theme.textPrimary
                        font.pixelSize: Theme.fontSizeMedium
                    }

                    MouseArea {
                        anchors.fill: parent
                        onClicked: {
                            // CdController.eject() — wired when available
                        }
                    }
                }

                // Search button placeholder (UI-14 — visible only on Spotify/Streaming)
                Rectangle {
                    width: Theme.touchTargetSmall
                    height: Theme.touchTargetSmall
                    radius: Theme.radiusSmall
                    color: "transparent"
                    visible: ReceiverState.currentInput === MediaSource.Streaming

                    Text {
                        anchors.centerIn: parent
                        text: "\uD83D\uDD0D" // magnifying glass
                        color: Theme.textPrimary
                        font.pixelSize: Theme.fontSizeMedium
                    }

                    MouseArea {
                        anchors.fill: parent
                        onClicked: UIState.setActiveView(ActiveView.SpotifySearch)
                    }
                }

                // Volume indicator placeholder (UI-12 — persistent small display)
                Text {
                    text: Math.round(ReceiverState.volume / 2) + "%"
                    color: Theme.textSecondary
                    font.pixelSize: Theme.fontSizeBody
                }

                // Mute button (UI-17)
                Rectangle {
                    width: Theme.touchTargetSmall
                    height: Theme.touchTargetSmall
                    radius: Theme.radiusSmall
                    color: ReceiverState.muted ? Theme.errorColor : "transparent"

                    Text {
                        anchors.centerIn: parent
                        text: ReceiverState.muted ? "\uD83D\uDD07" : "\uD83D\uDD0A"
                        font.pixelSize: Theme.fontSizeMedium
                    }

                    MouseArea {
                        anchors.fill: parent
                        onClicked: ReceiverController.toggleMute()
                    }
                }

                // Power button (UI-17)
                Rectangle {
                    width: Theme.touchTargetSmall
                    height: Theme.touchTargetSmall
                    radius: Theme.radiusSmall
                    color: ReceiverState.powered ? Theme.accent : "transparent"

                    Text {
                        anchors.centerIn: parent
                        text: "\u23FB" // power symbol
                        color: ReceiverState.powered ? Theme.textPrimary : Theme.textDimmed
                        font.pixelSize: Theme.fontSizeMedium
                    }

                    MouseArea {
                        anchors.fill: parent
                        onClicked: ReceiverController.setPower(!ReceiverState.powered)
                    }
                }

                // Time display (UI-16)
                Text {
                    id: timeDisplay
                    color: Theme.textSecondary
                    font.pixelSize: Theme.fontSizeBody

                    Timer {
                        interval: 60000
                        running: true
                        repeat: true
                        triggeredOnStart: true
                        onTriggered: timeDisplay.text = Qt.formatTime(new Date(), "h:mm AP")
                    }
                }
            }
        }
    }

    // Main content area below status bar
    Row {
        anchors.top: statusBar.bottom
        anchors.bottom: parent.bottom
        anchors.left: parent.left
        anchors.right: parent.right

        // Left panel: Input carousel (UI-03)
        Rectangle {
            id: leftPanel
            width: Theme.leftPanelWidth
            height: parent.height
            color: Theme.secondaryBg

            // InputCarousel placeholder — replaced in Plan 10-02
            Column {
                anchors.centerIn: parent
                spacing: Theme.spacingMedium

                Text {
                    text: "INPUT"
                    color: Theme.textSecondary
                    font.pixelSize: Theme.fontSizeSmall
                    anchors.horizontalCenter: parent.horizontalCenter
                }

                Text {
                    text: {
                        switch (ReceiverState.currentInput) {
                            case MediaSource.Streaming: return "Streaming"
                            case MediaSource.Phono: return "Phono"
                            case MediaSource.CD: return "CD"
                            case MediaSource.Computer: return "Computer"
                            case MediaSource.Bluetooth: return "Bluetooth"
                            case MediaSource.Library: return "Library"
                            default: return "None"
                        }
                    }
                    color: Theme.textPrimary
                    font.pixelSize: Theme.fontSizeLarge
                    anchors.horizontalCenter: parent.horizontalCenter
                }
            }
        }

        // Right panel: Active view (UI-03)
        Rectangle {
            id: rightPanel
            width: parent.width - Theme.leftPanelWidth
            height: parent.height
            color: Theme.primaryBg

            // View switching based on UIState.activeView
            // Placeholder content — replaced in Plan 10-03
            Text {
                anchors.centerIn: parent
                text: {
                    switch (UIState.activeView) {
                        case ActiveView.NowPlaying: return "Now Playing"
                        case ActiveView.LibraryBrowser: return "Library Browser"
                        case ActiveView.SpotifySearch: return "Spotify Search"
                        default: return "Now Playing"
                    }
                }
                color: Theme.textPrimary
                font.pixelSize: Theme.fontSizeXLarge
            }
        }
    }
}
```

- Left panel is fixed width for InputCarousel (placeholder, filled in Plan 10-02)
- Right panel fills remaining space for NowPlaying/LibraryBrowser/SpotifySearch (placeholder, filled in Plan 10-03)
- All colors and sizes reference Theme.qml tokens
