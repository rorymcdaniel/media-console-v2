**Progress ring for 4-second auto-select (UI-05):**
- When the user navigates to a new input (focus changes), start a 4-second timer
- Draw a circular progress ring around the focused item using Canvas or QtQuick.Shapes ShapePath with PathArc
- The ring fills clockwise over 4 seconds (use NumberAnimation on a progress property from 0 to 1)
- When the ring completes (progress reaches 1.0), call `ReceiverController.selectInput(focusedSource)`
- Tapping the focused item or pressing enter selects immediately (cancels timer)
- If user navigates away, reset the progress ring

**Auto-select timer implementation:**
```qml
property int focusedIndex: -1
property real autoSelectProgress: 0.0

Timer {
    id: autoSelectTimer
    interval: 40  // ~25fps update
    repeat: true
    onTriggered: {
        autoSelectProgress += (40 / 4000) // 4 seconds total
        if (autoSelectProgress >= 1.0) {
            autoSelectTimer.stop()
            ReceiverController.selectInput(sources[focusedIndex])
        }
    }
}

function focusInput(index) {
    focusedIndex = index
    autoSelectProgress = 0.0
    autoSelectTimer.restart()
}
```

**Dynamic accent color extraction:**
- When album art loads (Image.status === Image.Ready), use a Canvas element to sample the dominant color
- Draw the loaded image onto a small Canvas (1x1 or 10x10), read pixel data with getImageData()
- Average the RGB values and set Theme.dynamicAccent to the result
- Use Theme.dynamicAccent for: progress bar fill, accent highlights on controls

```qml
Canvas {
    id: colorExtractor
    width: 10; height: 10; visible: false

    function extract(imgSource) {
        var ctx = getContext("2d")
        ctx.drawImage(imgSource, 0, 0, 10, 10)
        var data = ctx.getImageData(0, 0, 10, 10).data
        var r = 0, g = 0, b = 0
        for (var i = 0; i < data.length; i += 4) {
            r += data[i]; g += data[i+1]; b += data[i+2]
        }
        var count = data.length / 4
        Theme.dynamicAccent = Qt.rgba(r/count/255, g/count/255, b/count/255, 1.0)
    }
}
```
