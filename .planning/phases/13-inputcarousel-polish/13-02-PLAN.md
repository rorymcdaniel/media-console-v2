    **Component structure:**

    ```
    Item (root, anchors.fill: parent, z: 500)
      ├── ListModel (sourceModel — unchanged from current)
      ├── properties: focusedIndex, autoSelectProgress
      ├── Timer (autoSelectTimer — adapted, calls carouselOverlay.hide() after select)
      ├── functions: sourceToIndex, indexToSource, focusInput, selectImmediately
      ├── function show() { carouselOverlay.show() }   ← exposed for left panel
      ├── Connections (UIState signals → show + navigate)
      ├── Connections (ReceiverState.currentInputChanged → sync index, no positionViewAtIndex)
      └── Rectangle (carouselOverlay)
            ├── color: Qt.rgba(0, 0, 0, 0.70)
            ├── visible: false (starts hidden)
            ├── opacity: 0.0
            ├── Behavior on opacity { NumberAnimation animMedium InOutQuad }
            ├── function show() { visible = true; opacity = 1.0 }
            ├── function hide() { autoSelectTimer.stop(); autoSelectProgress = 0.0; opacity = 0.0; hideTimer.restart() }
            ├── Timer (hideTimer, interval: Theme.animMedium, sets visible: false)
            ├── MouseArea (anchors.fill: parent → carouselOverlay.hide())
            └── PathView (carousel, on top of MouseArea via z or placement order)
                  ├── anchors.centerIn: parent
                  ├── width: parent.width; height: 420 (card height + padding)
                  ├── model: sourceModel
                  ├── snapMode: PathView.SnapToItem
                  ├── preferredHighlightBegin: 0.5
                  ├── preferredHighlightEnd: 0.5
                  ├── highlightRangeMode: PathView.StrictlyEnforceRange
                  ├── pathItemCount: sourceModel.count  (all 6, prevents wrap artifacts)
                  ├── interactive: true
                  ├── path: Path { startX, startY → PathLine { x, y } }
                  ├── onMovementStarted: autoSelectTimer.stop()
                  ├── onMovementEnded: restart timer if pending selection
                  ├── onCurrentIndexChanged: root.focusInput(currentIndex)
                  └── delegate: Item (card with scale/opacity depth effect)
    ```

    **PathView path dimensions (Claude's Discretion, per research recommendations):**
    - Card size: 220px wide × 320px tall
    - Card spacing: 280px center-to-center
    - Path startX: -140 (center of leftmost card when left edge is at 0; PathView is 1920px wide)
    - Path endX: 1920 + 140 = 2060
    - Path Y: carousel.height / 2
    - PathView height: 380px (320px card + 30px top/bottom breathing room)

    **Delegate depth effect (distFromCenter from PathView.view.offset):**
    ```qml
    property real distFromCenter: {
        var n = sourceModel.count          // 6
        var offset = PathView.view.offset
        var raw = index - (n - offset) % n
        if (raw > n / 2)  raw -= n
        if (raw < -n / 2) raw += n
        return Math.abs(raw)
    }
    property real itemScale:   distFromCenter < 0.5 ? 1.0 : distFromCenter < 1.5 ? 0.72 : 0.45
    property real itemOpacity: distFromCenter < 0.5 ? 1.0 : distFromCenter < 1.5 ? 0.55 : 0.25

    scale:   itemScale
    opacity: itemOpacity
    Behavior on scale   { NumberAnimation { duration: Theme.animMedium; easing.type: Easing.OutCubic } }
    Behavior on opacity { NumberAnimation { duration: Theme.animMedium; easing.type: Easing.OutCubic } }
    ```

    **Delegate card content:** Icon circle (96px diameter, Theme.accent fill when isCurrent) + iconText (36px) + labelText below (Theme.fontSizeMedium, bold when isCurrent). Progress ring Canvas on focused card (carry forward from current implementation — adapts to 96px icon circle). MouseArea fills delegate: onClicked calls root.selectImmediately(index) then carouselOverlay.hide().

    **UIState Connections block:**
    ```qml
    Connections {
        target: UIState
        function onInputNextRequested() {
            carouselOverlay.show()
            carousel.incrementCurrentIndex()
            root.focusInput(carousel.currentIndex)
        }
        function onInputPreviousRequested() {
            carouselOverlay.show()
            carousel.decrementCurrentIndex()
            root.focusInput(carousel.currentIndex)
        }
        function onInputSelectRequested() {
            if (!carouselOverlay.visible) {
                carouselOverlay.show()
            } else {
                root.selectImmediately(carousel.currentIndex)
                carouselOverlay.hide()
            }
        }
    }
    ```

    **ReceiverState sync:** Replace the old positionViewAtIndex() (ListView-only) with carousel.currentIndex = idx assignment:
    ```qml
    Connections {
        target: ReceiverState
        function onCurrentInputChanged() {
            var idx = root.sourceToIndex(ReceiverState.currentInput)
            if (idx >= 0) {
                root.focusedIndex = idx
                root.autoSelectProgress = 0.0
                autoSelectTimer.stop()
                carousel.currentIndex = idx   // PathView: use currentIndex, not positionViewAtIndex
            }
        }
    }
    ```

    **Critical ordering in show():** Set `visible = true` BEFORE `opacity = 1.0` — otherwise the Behavior on opacity never fires (Qt skips animations on invisible items).

    **Critical: stop timer in hide():** Call `autoSelectTimer.stop()` and reset `autoSelectProgress = 0.0` inside hide() to prevent stale auto-select after carousel dismissal.

    **Imports needed:** `import QtQuick`, `import MediaConsole 1.0` (for UIState, ReceiverState, ReceiverController, MediaSource singletons).
