### Missing Connections (5)

**Priority 1 — Runtime failure:**
- `FlacLibraryController::playTrack()` is `public` but not `Q_INVOKABLE` — QML `LibraryBrowser.qml` line 525 call silently fails
- **Fix:** Add `Q_INVOKABLE` to declaration in `src/library/FlacLibraryController.h`

**Priority 2 — Feature unreachable:**
- `LocalPlaybackController::audioRecoveryFailed()` emitted but never connected to `UIState::setAudioError()` in AppBuilder
- **Fix:** Add `connect()` in `src/app/AppBuilder.cpp`

**Priority 3 — Feature incomplete:**
- `CdController` not registered as QML singleton in `main.cpp`; eject button `onClicked` empty; no CD track browser
- **Fix:** `qmlRegisterSingletonInstance` in `main.cpp`, add `Q_INVOKABLE` to CdController methods

**Priority 4 — Feature gap:**
- `PlaybackRouter::seek()` has empty `break` for CD and Library — `LocalPlaybackController::seek()` is unreachable
- **Fix:** Route seek calls through to `LocalPlaybackController` in `PlaybackRouter`

**Priority 5 — Correctness concern:**
- `ReceiverController::parseResponse()` updates `ReceiverState.volume` directly, bypassing `VolumeGestureController::onExternalVolumeUpdate()` — gesture reconciliation during active gestures may snap back
- **Fix:** Connect `ReceiverController` volume updates through `VolumeGestureController`

## Conclusion

The v1.0 milestone has strong coverage in the later phases (7-10), which have detailed verification reports and passing integration. The earlier phases (1-6) were executed before verification was standardized, leaving documentation gaps. The ROADMAP confirms all plans were executed with commit hashes, and REQUIREMENTS.md marks most as Complete — the primary gap is the verification trail, not the code.

**Critical blockers (must fix):**
1. FlacLibraryController.playTrack() not Q_INVOKABLE — FLAC library playback broken from QML
2. audioRecoveryFailed not wired to UIState — AudioErrorDialog unreachable
3. CdController not registered as QML singleton — CD track selection and eject unavailable from UI
4. PlaybackRouter.seek() empty for local sources — seek broken for CD/FLAC
5. VolumeGestureController reconciliation bypass — volume may snap during encoder use

**Documentation gaps (non-blocking but should address):**
- 5 phases missing VERIFICATION.md (Phases 1, 2, 4, 5, 6)
- Phase 6 missing all SUMMARY files
- REQUIREMENTS.md traceability stale for FOUND and CD groups
